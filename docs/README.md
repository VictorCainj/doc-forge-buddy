# Documentação Consolidada

A pasta `docs` concentra os materiais de apoio ao projeto. Subdivisões principais:

- `reports/`: relatórios, resumos executivos e análises de auditoria.
- `guides/`: guias práticos e instruções de implantação.
- `tasks/`: registros de tarefas concluídas e acompanhamentos operacionais.
- `lists/`: listagens de referência, como inventários de arquivos implementados.
- `validations/`: validações e verificações específicas executadas no projeto.

Arquivos que permanecem direto na raiz de `docs` são referência geral ou índices. Ao adicionar novos documentos, escolha a subpasta apropriada ou crie uma nova categoria quando necessário.
